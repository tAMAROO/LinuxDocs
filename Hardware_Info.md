## Hardware information Commands


### System Uptime:

```$uptime```

### System Information:

```$uname```

### CPU Information:

```$lscpu```

### CPU Information 2:

```$sudo dmidecode -t bios```

### Memory Usage:

```$free -h```

### System Process Utility:

```$top```

### PCI Infomation:

```$lspci```

### USB Information:

```$lsusb```

### Disk Space:

```$df -h```

### List Block Devices:

```$lsblk```
